# Papers table Category order plan

## Summary

Move the frontend Papers table `Category` column to the left of `ID`.

## Implementation

- Change the Papers table leading column order from `Status`, `ID`, `Category`, `Title` to `Status`, `Category`, `ID`, `Title`.
- Bump the Papers grid persistence id to avoid stale saved column ordering.

## Tests

- Run frontend build.
- Run `git diff --check`.
