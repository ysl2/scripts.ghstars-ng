# 详情页 arXiv ID 与标题复制

## 需求对齐

- 将论文详情抽屉顶部小字从 `Paper details` 替换为当前论文的 `arxiv_id`。
- 顶部 `arxiv_id` 文本支持点击复制，复制内容为 `selectedPaper.arxiv_id`。
- 顶部标题文本支持点击复制，复制内容为 `selectedPaper.title`。
- 点击复制后显示短暂提示；复制失败时也给出提示。
- 保留详情字段里的 arXiv 外链跳转语义，不把现有外链改成复制操作。

## 实现计划

1. 在 `frontend/src/App.tsx` 增加轻量复制反馈状态、清理定时器和复制处理函数。
2. 将论文详情抽屉顶部的 `arxiv_id` 与标题渲染为可点击的文本按钮，并接入复制处理函数。
3. 在 `frontend/src/App.css` 增加可点击文本和复制提示样式，保持现有详情抽屉视觉风格。
4. 验证：运行前端构建和 lint。
